### `Run the project on localhost`
Open two terminal and run the following commands in root directory in each of them
Client Terminal
1. Install dependencies for client
```sh
$ cd client && npm i
```
2. Run client
```sh
$ npm run dev
```

Server Terminal
1. Install dependencies for server
```sh
$ cd server && npm i
```
2. Run server
```sh
$ npm run dev
```
